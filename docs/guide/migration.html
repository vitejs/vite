output.experimentalMinChunkSize

In Vite v7, the output.experimentalMinChunkSize option was used to set the minimum chunk size. In Vite v8, this option has been deprecated in favor of rolldownOptions.output.codeSplitting.minSize.

Example:

import { defineConfig } from 'vite';

export default defineConfig({
  build: {
    rollupOptions: {
      output: {
        codeSplitting: {
          minSize: 2000
        }
      }
    }
  }
});
